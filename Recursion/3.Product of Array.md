# Product of Array

Write a function calledÂ `productOfArray`Â which takes in an array of numbers and returns the product of them all.

Examples:
* `productOfArray([1,2,3]) // 6`
* `productOfArray([1,2,3,10]) // 60`

---

### Solution
```js
function productOfArray(arr) {
  if(!arr.length) return 1;
  return arr[0] * productOfArray(arr.slice(1));
}
```
